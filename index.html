Project Work

Just over halfway through University of Arizona's Full Stack Development Boot Camp, I have successfully completed thirteen individual projects and one group project. Four of my projects are featured below with links to the project repositories and live pages on GitHub:


Picky Eater Recipe App

This was a group project to develop a recipe application. The project scope sought to provide a resource for people with food allergies and sensitivities to find healthy recipes matching their dietary restrictions. The application was to save the user's last search in local storage.

Visit page Project code

Project Day Scheduler

This was a planner for a project manager and her team. The project scope sought to provide an app allowing users to save notes to each hour of the day and effectively manage important project events. The app features dynamically updated HTML and CSS powered by jQuery.

Visit page Project code

JavaScript Code Quiz

This was a project for web development students to test their knowledge of JavaScript fundamentals. The app was to provide a timed quiz which stores high scores enabling students to gauge their progress with their peers and render responsively across a variety of devices.

Visit page Project code

Eat Da Burger

This project was a burger logger developed using MySQL, Node, Express, Handlebars and a homemade ORM (yum!). An MVC design pattern was followed using Node and MySQL to query and route data in the app, and Handlebars to generate the HTML.

Freelance Passion Project

As for my creative side, I am an amateur photographer and painter. A couple of years ago I started creating art to memorialize and celebrate pets. It began when a friend on Instagram reached out to me for ideas on how she could memorialize a family friend's pup that had just passed. The photos I was given of the sweet pup were low resolution so I decided to overlay a watercolor of her over a photo background of butterflies and sunflowers. The family loved the piece so much they framed it! Click the 'See Pup!' button to see the final art piece.
